# Mark's Plumbing — Business Website v2

A responsive business website for a San Antonio, TX plumber. Dark industrial aesthetic focused on driving phone calls and displaying customer reviews.

## 🔧 Built With

- HTML5
- CSS3 (custom properties, grid, flexbox, animations)
- Vanilla JavaScript (no frameworks or dependencies)

## ✨ Features

- Floating call button (appears after scrolling past hero)
- Dedicated reviews section with featured review card
- Mobile hamburger navigation
- Scroll-triggered fade-in animations (Intersection Observer API)
- Contact form with client-side validation
- 24/7 emergency strip with pulse animation
- Fully responsive — mobile, tablet, desktop

## 📁 File Structure

```
marks-plumbing/
├── index.html    # Page structure and content
├── style.css     # All styling and responsive layout
├── script.js     # Floating button, nav, form, scroll animations
└── README.md
```

## 🚀 Getting Started

Open `index.html` in a browser — no build tools or dependencies needed.
