This folder is for image assets used by the website.

How to use this folder:
- Add your logo file here, for example: logo.svg or logo.png
- Add section photos here, for example: team-mark.jpg or service-drain.jpg
- Keep all image files in this folder and reference them from index.html with src="images/filename.ext"

Example HTML reference:
<img src="images/logo.svg" alt="SA GoodMen Plumbing logo" />

Tip:
- Add your real logo file to this folder and reference it from index.html.

You can edit this file to keep track of image names or to leave yourself reminders.
