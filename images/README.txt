This folder is for image assets used by the website.

How to use this folder:
- Add your logo file here, for example: logo.svg or logo.png (the nav will display it automatically)
- Add service pictures here, for example: service-leak.jpg, service-drain.jpg, etc. (6 files for the services section)
- Add gallery images here, for example: gallery-1.jpg, gallery-2.jpg, etc. (6 files for the gallery section)
- Keep all image files in this folder and reference them from index.html with src="images/filename.ext"

Example HTML reference:
<img src="images/logo.svg" alt="SA GoodMen Plumbing logo" />

Tip:
- Add your real logo file to this folder and it will appear in the top left nav.
- Replace the placeholder filenames with your actual image files.

You can edit this file to keep track of image names or to leave yourself reminders.
